WanderStay is an Airbnb-style web application built using Node.js, Express, MongoDB, and EJS.
It allows users to browse property listings, manage stays, and maintain secure user sessions.

Features

User authentication & authorization
Create, edit, delete property listings
Category & country-based filtering
Persistent sessions using MongoDB
Server-side rendering with EJS

 Tech Stack

Frontend: EJS, Bootstrap

Backend: Node.js, Express

Database: MongoDB, Mongoose

Sessions: express-session, connect-mongo
